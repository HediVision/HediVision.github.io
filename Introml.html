    Intro

    One of main application of machine learning is Classification. The classification is the process of predicting the object category with given sample data. The classification is achieved either by constructing a model for each class (generative) or by finding the decision boundary between classes (discriminative) in the sample space (Figure 1).

    Figure 1. The generative approach onthe left assigns the new sample, grey triangle, to the closest class model, while the discriminative classification, right side, used direct mapping by finding the boundaries between the classes C1, and C2. The red line is classifying plane.

    Red and blue markers in Figure 1 are the feature vectors. A feature vector is an n-dimensional vector of numerical features that represent some property of object.
